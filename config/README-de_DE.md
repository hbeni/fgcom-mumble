## Konfigurationsoptionen

FGCom-Mumble unterstützt umfangreiche Konfiguration über verschiedene Methoden:

1. **Server-Konfiguration**: `~/.fgcom-mumble.ini`
2. **Client-Konfiguration**: Mumble-Plugin-Einstellungen
3. **Radio GUI-Konfiguration**: Java-Anwendungseinstellungen

Siehe die Hauptprojektdokumentation für detaillierte Konfigurationsanweisungen.
