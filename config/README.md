# FGCom-Mumble Configuration

## Configuration Files

This directory contains configuration files and documentation for FGCom-Mumble.

### Files

- `Readme.architecture.md` - System architecture documentation
- `README.md` - This file

## Configuration Options

FGCom-Mumble supports extensive configuration through various methods:

1. **Server Configuration**: `~/.fgcom-mumble.ini`
2. **Client Configuration**: Mumble plugin settings
3. **Radio GUI Configuration**: Java application settings

See the main project documentation for detailed configuration instructions.
